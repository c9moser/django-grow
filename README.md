# django-grow

Grow App for manage your grows.

Currently you can manage your seeds in stock and create growlogs. Support for managing
motherplants and the resulting clones will be added in the future. A REST-webapi interface
and support for sensors like temperature and humidity is also planned. Automatically updating
the sensor data will be supported by the webapi.

You can see yourself what this app does and how the devlopment progresses on
[GrowBook.eu](https://growbook.eu).

You can reach me (and other developers if i find some contributors) under the email
address *dev at growbook.eu*.

Docker/Podman images will be released when development progresses.
